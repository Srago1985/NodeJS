# NodeJS

Monorepo for NodeJS lessons.

## Structure

Each lesson should be placed in a separate root folder, for example:

- lesson_2
- lesson_3-NodeJS
- lesson_4-NodeJS
- lesson_5

## Adding a new lesson

1. Create a new root folder in this repository (for example, lesson_5).
2. Add lesson files inside that folder.
3. Commit and push to main.

This keeps every lesson as an isolated folder inside the NodeJS repository.
